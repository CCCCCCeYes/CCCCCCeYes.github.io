Install package control and emmet in sublime text3

Posted on 2022-09-11

检查一下能否直接打开该网页

Installation - Package Control

手动安装

按照指示下载，得到一个后缀为.sublime-package的文件

复制到自己sublime text 3的安装目录中，例如我的是D:\Sublime Text 3\Packages

打开后复制文件于其中，重启后Preferences\Package Control

稍等片刻可以看到 Package Control: Install Package

点击后再次稍等片刻

输入emmet 可以看到看到左下角显示正在安装

调整右下角为HTML

html:5 +Tab键即可快捷生成模板
